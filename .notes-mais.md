   - Sampling defaults: `temperature`, `top_p`, `top_k`, `presence_penalty`
   - `chat_template_kwargs`: `preserve_thinking`, `enable_thinking`
   - Exposed on Monitor page per loaded model
   - See `docs/model-config-plan.md`

2. **Chat history persistence**
   - **PROBLEM:** `sessionStore.ts` is pure in-memory
   - **IMPACT:** History lost on every page refresh
   - **FIX NEEDED:** Persist to `localStorage` (messages + selectedModel + systemPrompt)

3. **Backend crash detection**
   - Periodic health ping from server to loaded backends
   - Mark model as "error" on failure
   - Frontend notification when backend crashes

4. **HuggingFace API token config**
   - Settings UI field + persisted to `settings.json`
   - Required for gated models (Llama, Mistral, etc.)

## Architecture Notes

- **Server:** FastAPI with process manager (start/stop backends)
- **Port Ranges:** 8081-8099 (GGUF), 8100-8119 (MLX)
- **Proxy:** Routes OpenAI requests to correct backend
- **Frontends:** React + Vite + Tailwind, TanStack Query
- **Telemetry:** SQLite storage, TTFT + tokens/sec + counts
